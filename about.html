サイトについて

1. フリー素材ポータルについて

フリー素材ポータルは、「クリエイターが安心して使えるフリー素材を、最速で見つけられるようにする」をミッションに運営しているポータルサイトです。

Web制作、グラフィックデザイン、動画編集、ゲーム開発など、クリエイティブ制作に必要なフリー素材サイトを221サイト以上掲載しています。写真・イラスト・音楽・動画・アイコン・フォントなど13カテゴリに整理し、「商用利用OK」「クレジット不要」「登録不要」などのフィルターでワンクリックで絞り込めます。2026年に運営を開始しました。

2. 掲載サイトの選定基準

当サイトに掲載するフリー素材サイトは、以下の基準で選定しています。

  • 公式サイトが存在し、安定して運営されていること
  • 利用規約が明確に記載されていること
  • ダウンロードが実際に可能であること（リンク切れでないこと）
  • 悪質な広告やマルウェアの報告がないこと

掲載情報は定期的に確認・更新していますが、各サイトの利用規約は予告なく変更される場合があります。必ず公式サイトで最新情報をご確認ください。

3. 運営者情報

運営者: freesozo編集部

Web制作・デザイン分野で活動するクリエイターチームが運営しています。

お問い合わせ: GitHub Issues よりお願いいたします。

4. 免責事項

  • 当サイトはアフィリエイトプログラムに参加しています。当サイト経由で商品・サービスをご購入いただいた場合、紹介料を受け取ることがあります。これによりユーザーに追加の費用が発生することはありません。
  • 掲載情報の正確性について最善を尽くしていますが、その内容を保証するものではありません。
  • 各サービスのご利用は自己責任でお願いいたします。当サイトの情報を利用することによって生じたいかなる損害についても、責任を負いかねます。
